형태소분석기(Morpheme Analysor)
1. KAIST의 한나눔 형태분석기 라이브러리 활용
   http://kldp.net/projects/hannanum/
   
2. 설치
   2.1 한나눔 형태분석기 라이브러리 다운로드
       http://kldp.net/projects/hannanum/download 에서 
       JHanNanum-0.8.4-ko.zip 다운로드
       
   2.2 압축해제
       GUIDemo : 형태소 분석기 데모 프로그램
       JHanNanum : 형태소 분석기 프로젝트 디렉토리
                   하위 \doc에 Java API 참고
       conf.zip : 플러그인 환경설정 파일
       data.zip : 시스템 사전, 기분석 사전, 사용자 사전, 태그셋, 결합정보 등 포함
       jhannanum-0.8.3.jar : 라이브러리
       
   2.3 프로젝트 내 jhannanum-0.8.3.jar 추가
          클래스 패스 루트 conf 팩키지 생성 후 conf.zip 압축 해제 추가 
          클래스 패스 루트 data 팩키지 생성 후 data.zip 압축 해제 추가
