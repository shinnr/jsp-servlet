GrepCode
라이브러리(*.class) 역컴파일 플러그인

1. 설치
   Help -> Install New Software -> [Add] 클릭 -> name : grepcode
                                               location : http://repository.grepcode.com/java/ext-eclipse/
    
2. 이클립스 상단 GC Search 툴바 생성 확인

3. 활용
     라이브러리의 특정 팩키지의 특정 클래스 더블클릭으로 해당 class 파일의 GC뷰를통한 소스코드 확인
