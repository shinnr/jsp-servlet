Stan4j(Structure Analysis for JAVA)
[ http://stan4j.com/ ]

* 팩키지 분석 툴로서 팩키지 내 구조 및 클래스간의 Cohesion(응집도), 결합도(Coupling) 측정 툴.
* 특정 팩키지 내 500개 이상의 클래스 대상 분석을위해서는 라이센스 구매 필요
 
1. 설치
   Help -> Install New Software -> http://update.stan4j.com/ide 입력 후 설치

2. 활용
   
   2.1 대상 팩키지 선택
   2.2 Run As -> Structure Analysis 실행으로 Stan Perspective 전환 및 활용
